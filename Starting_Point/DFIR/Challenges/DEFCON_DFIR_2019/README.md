<H1>Defcon DFIR CTF 2019</H1>
<hr>

<H2>What is it></H2>
<p>
The Defcon DFIR CTF 2019 is a CTF centred around DFIR, it has 5 series of challenges each working on a specific area of forensics. they are:
<br>
- DFA Crypto Challenge
- Deadbox Forensics
- Linux Forensics
- Memory Forensics
- Triage VM Questions
