<H2>Challenge</H2>
<details>
    <summary></summary>
<p></p>
What is the value of y2?
<p></p>

```
with
  t1(x1, x2) as (select 1, 2),
  t2(y1, y2) as (
    select x1  2, x2  2
    from t1
  )
select *
from t1, t2
```

Flag will be in the format FLAG{#}
